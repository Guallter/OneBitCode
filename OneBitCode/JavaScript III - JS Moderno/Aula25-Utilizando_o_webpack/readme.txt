
Comandos:
npm i --save-dev webpack webpack-cli

npm i --save-dev webpack: Instalação padrão
npm i --save-dev webpack-cli: Interface de linha de comando para executar no terminal



ao dar o comando:

-npx webpack 

ele o executa criando o arquivo main.js de forma otimizada (em uma unica linha) removendo os espaços
desnecessarios, e ele remmove o vinculo obrigatorio com o modulo node_modules, nessa etapa de "empacotamento"
