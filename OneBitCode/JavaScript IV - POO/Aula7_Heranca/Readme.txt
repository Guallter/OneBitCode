Recursos que herdam todos os atributos e metodos de outra classe.
Exemplo pratico:

No sistema teremos:
-Administardor
-Editor
-Apenas visualizar

Todos eles serão usuarios e terão informações como:
-Nome
-Email
-Senha

Sendo assim, podemos criar a classe USUARIO com os atributos que as três tem em comum,
e as 3 outras classes vão herdar os atributos e metodos dela.
